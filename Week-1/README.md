# Week 1 - Python Full Stack Engineer Training

This week introduces core Python problem-solving topics and common software design patterns.

## Folder Guide

- `DesignPatterns/` contains beginner-friendly examples of SOLID principles and the classic creational, structural, and behavioral patterns.
- `DSA/` contains algorithm analysis, array basics, and searching/sorting examples.

## How to Run

Run any Python file directly:

```bash
python DesignPatterns/solid_principles.py
python DesignPatterns/creational_patterns.py
python DesignPatterns/structural_patterns.py
python DesignPatterns/behavioral_patterns.py
python DesignPatterns/mvc.py
python DSA/analysis_of_algorithms.py
python DSA/sorting.py
python DSA/arrays.py
python DSA/searching.py
```

## Learning Goals

- Understand how to organize code with simple design patterns.
- Learn the basics of algorithm complexity.
- Practice array operations, sorting, and searching.
- Read and run small, focused examples before moving to larger applications.
