CSS3 Layouts: Flexbox vs. Grid

Modern layouts use Flexbox (best for 1-dimensional item alignment in a single row or column) and Grid (best for 2-dimensional grid layouts, dealing with rows and columns simultaneously).

1. CSS Flexbox (1D Layout)

Flexbox works on a single axis (either row or column). Below, items wrap and share space dynamically using flex-grow.

Common Flex Attributes:

  • display: flex;: Initializes flex context on parent.
  • justify-content:: Align items along the main axis.
  • align-items:: Align items along the cross axis.
  • flex-grow:: Ratios on how children expand to consume empty space.

2. CSS Grid (2D Layout)

CSS Grid handles both horizontal columns and vertical rows. Elements can be assigned explicit grid coordinates.

Common Grid Attributes:

  • display: grid;: Initializes grid context.
  • grid-template-columns:: Defines column widths (e.g., repeat(3, 1fr), 200px 1fr 2fr).
  • grid-column / grid-row:: Defines starting/ending track coordinates for child elements (e.g., span 3, 1 / 3).
  • gap:: Setting spacing between columns/rows directly.
