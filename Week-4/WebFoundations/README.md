# Web Foundations

This module covers the core pillars of web development: **HTML5**, **CSS3**, and modern **JavaScript (ES6+)**. Regardless of the frontend framework you use (React, Angular, Vue), they all compile down to these three foundational technologies.

## Contents

### 1. [semantic_html.html](file:///c:/Personal/Projects/CTS-DN/Week-4/WebFoundations/semantic_html.html)
- **Topic**: HTML5 Semantic elements.
- **Concepts**: Layout structure using `<header>`, `<nav>`, `<main>`, `<article>`, `<section>`, `<aside>`, and `<footer>` instead of unsemantic `<div>` wrapping.
- **Goal**: Write structured, clean, search-engine-optimized, and accessible markup.

### 2. [css_layout.html](file:///c:/Personal/Projects/CTS-DN/Week-4/WebFoundations/css_layout.html)
- **Topic**: CSS3 Flexbox and Grid.
- **Concepts**: Real-world comparison and demonstration of 1D styling (Flexbox) vs. 2D styling (Grid).
- **Goal**: Master aligning elements, responsive sizing, grids, and auto-placements without relying on heavy external CSS libraries.

### 3. [modern_javascript.js](file:///c:/Personal/Projects/CTS-DN/Week-4/WebFoundations/modern_javascript.js)
- **Topic**: Modern JavaScript (ES6+).
- **Concepts**: Variables scoping, Arrow functions, Asynchronous flows (Promises, Async/Await), and modular file structures.
- **Goal**: Understand modern execution patterns in JavaScript that frontend frameworks rely on for state updates and asynchronous operations.
